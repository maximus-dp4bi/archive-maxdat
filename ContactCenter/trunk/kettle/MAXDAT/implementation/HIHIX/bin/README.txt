INSTRUCTIONS FOR RUNNING HIHIX JOBS
-----------------------------------

If you'd like to process data files containing the previous day's data, please run the following command:

run_load_contact_center.sh

If you'd like to process data files containing data for days other than the previous day, please run the following command
where the startDate and endDate are in the format yyyy/mm/dd:

run_batch_load_contact_center.sh [startDate] [endDate]
